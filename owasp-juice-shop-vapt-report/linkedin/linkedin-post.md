# LinkedIn Post Draft

I completed a practical Web Application VAPT project using OWASP Juice Shop as a local lab target.

For this project, I deployed Juice Shop with Docker and performed reconnaissance, automated scanning, and manual verification using tools such as Nmap and Nikto.

Key areas covered:

- SQL injection authentication bypass
- Exposed directory listing
- Confidential document exposure
- Verbose error / stack trace disclosure
- Missing security headers
- Overly permissive CORS configuration
- Robots.txt sensitive path disclosure

The goal was not only to find vulnerabilities, but also to document them professionally with evidence, impact, severity, and remediation steps, similar to a real VAPT report.

This project helped me strengthen my practical understanding of OWASP Top 10 vulnerabilities, manual verification, scanner validation, and security reporting.
