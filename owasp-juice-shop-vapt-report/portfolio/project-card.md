# Portfolio Project Card

## Title

OWASP Juice Shop VAPT Report

## Short Description

Performed a web application vulnerability assessment on OWASP Juice Shop using Nmap, Nikto, and manual testing. Identified and documented SQL injection authentication bypass, confidential document exposure, exposed directory listing, verbose error disclosure, missing security headers, and CORS misconfiguration.

## Tools

Docker, Nmap, Nikto, OWASP Juice Shop, Manual Testing, Browser DevTools

## Highlights

- Deployed OWASP Juice Shop locally with Docker.
- Performed reconnaissance and automated web scanning.
- Manually verified exposed files and sensitive path disclosure.
- Demonstrated SQL injection authentication bypass.
- Created a professional VAPT report with impact and remediation guidance.
